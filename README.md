# PASSAPORTO ROBOT

This app goes to the website of Passport Request to check wheter there is a date available for an appointment within the parameters specified.

## Prerequisites
~~~
npm install pkg --global
npm install
~~~

## Development Server

~~~
npm run start
~~~

## Build
~~~
npm run build
~~~
